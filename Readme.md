### Connected Blocks
- Glass, glass pane, stained glass (all colours), stained glass pane (all colours), tinted glass
- Sea lanterns, shroomlights, froglights
- Bookshelves
- Blocks of ores (copper, diamond, emerald, gold, iron, lapis lazuli, netherite and redstone)
- Ores (copper, diamond, emerald, gold, iron, lapis lazuli, netherite and redstone)


#### Dependency
- [Continuity](https://modrinth.com/mod/continuity) mod or optifine.
